## Auto push

`scripts/start-auto-push.bat` を起動すると、ファイル変更を検知して自動で `git add -A`、`git commit`、`git pull --rebase`、`git push` を実行します。

共同編集で衝突した場合は `.auto-push.log` にエラーを書いて停止します。衝突を直してから、もう一度 `scripts/start-auto-push.bat` を起動してください。
