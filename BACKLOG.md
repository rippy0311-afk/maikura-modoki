# 今後のアップデート候補

作業中に見つかった未解決の課題。着手したら該当項目を消すか、コミットハッシュを添えて履歴に残す。
確認日: 2026-07-20

---

## 優先度: 中

### 1. クラフトが簡易版

現在は「材料が揃っていればボタンで作れる」形式。
2x2 グリッドに実際にアイテムを置く本格版にするなら、レシピの形状定義とドラッグ&ドロップの実装が必要。
作業台の 3x3 グリッドを入れる場合はさらに拡張が必要。

---

## 優先度: 低

### 2. テストハーネスが残っていない

WebGL をスタブして `main.js` をそのまま動かし、破壊・取得・クラフト・UI表示を検証する仕組みを一時的に作ったが、リポジトリには入れていない。
これがあると「アイテムが取れない」類の問題を数分で切り分けられる。

### 3. アイコンをさらに良くする

現在はテクスチャアトラスから側面タイルを切り出して使用している。
これ以上を求めるなら、CC0 素材の導入か、`blocks.js` の `makeAtlasCanvas()` を拡充する。

Minecraft のアセットは使えない。
本リポジトリは public かつ GitHub Pages で公開配信しているため、アセットを入れた時点で再配布に当たり、Mojang の EULA に抵触する。
